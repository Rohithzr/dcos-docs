# Description
Kill the job.

# Usage

```bash
dcos job kill [OPTION]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `--help, h`   |             |  Print usage. |
| `--info`   |             |  Print a short description of this subcommand. |
| `<job-id>`   |             |  The job ID. |
| `run-id`   |             |  The job run ID. |
| `--all`   |             |  Kill all job runs. |
| `--version, v`   |             | Print version information. |
