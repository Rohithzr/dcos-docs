# Description
This command validates changes to the configuration file.

# Usage

```bash
dcos config validate [OPTION]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `--help, h`   |             |  Print usage. |
| `--info`   |             |  Print a short description of this subcommand. |
| `--version`   |             |  Print version information. |
